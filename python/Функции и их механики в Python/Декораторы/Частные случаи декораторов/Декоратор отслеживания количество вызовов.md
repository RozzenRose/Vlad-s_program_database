#python #декоратор 

Иногда полезно иметь декоратор, который может отслеживать состояние вызова функции. Создадим декоратор, который подсчитывают, сколько раз вызывается функция. Для сохранения состояния счетчика будем использовать пользовательский атрибут функции

```python
import functools

def counter(func):
    @functools.wraps(func)
    def wrapper(*args, **kwargs):
        wrapper.num += 1
        print(f'Вызов {func.__name__}: {wrapper.num}')
        val = func(*args, **kwargs)
        return val
    wrapper.num = 0
    return wrapper

@counter
def greet(name):
    return f'Hello {name}!'

print(greet('Timur'))
print(greet('Ruslan'))
print(greet('Arthur'))
print(greet('Gvido'))
```
```
Вызов greet: 1
Hello Timur!
Вызов greet: 2
Hello Ruslan!
Вызов greet: 3
Hello Arthur!
Вызов greet: 4
Hello Gvido!
```
