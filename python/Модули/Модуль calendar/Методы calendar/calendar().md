#calendar #Python #метод #calendar 


Вернет многострочную строку, содержащую календарь на весь год
`calendar(year, w=2, l=1, c=6, m=3)`
`year` - год
`w` - ширина столбца даты (необязательный)
`l` - количество строк, отводимое на одну неделю (необязательный)
`c` - количество пробелов между столбцами месяца (необязательный)
`m` - количество столбцов (необязательный)
```python
import calendar

print(calendar.calendar(2021))
```

Можно русифицировать словарь при помощи локализации [[Модуль locale]]
```python
import calendar, locale

locale.setlocale(locale.LC_ALL, 'ru_RU.UTF-8')

print(calendar.calendar(2022, m=4))
```
