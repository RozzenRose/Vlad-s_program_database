#calendar #Python #метод #promonth #prcal


`prmonth()` - напечатает календарь указанного месяца, который возвращает функция [[month()]]
`pecal()` - напечатает календарь года, который возвращает функция [[calendar()]]

Вот методы `month()` и `calendar()` просто возвращают тебе строчки с календарями, а эти и два их печатают. Если запринтить `month()` и `calendar()` получится тоже самое

Не спрашивай нахуй они тогда вообще нужны!
```python
import calendar

calendar.prmonth(2021, 9)
calendar.prcal(2021)
```
эквивалентен коду:
```python
import calendar

print(calendar.month(2021, 9))
print(calendar.calendar(2021))
```
